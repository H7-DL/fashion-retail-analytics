# Fashion Retail Sales Performance Dashboard

Analisis data penjualan retail fashion untuk optimasi stok dan strategi bisnis.

## Project Overview

Proyek ini menganalisis **2,750 transaksi** penjualan fashion retail (Oktober 2022 - September 2023) untuk mengidentifikasi:
- Tren penjualan musiman
- Produk terlaris (fast-moving items)
- Perilaku konsumen berdasarkan metode pembayaran

## Business Questions

1. **Kapan waktu puncak penjualan terjadi?**
2. **Produk apa yang paling menghasilkan revenue?**
3. **Apakah metode pembayaran mempengaruhi nilai transaksi?**

## Tools & Technologies

- **Database:** PostgreSQL (Aiven Cloud)
- **SQL Client:** DBeaver
- **Data Cleaning:** Microsoft Excel
- **Visualization:** Microsoft Power BI

## Key Findings

### 1. Tren Penjualan Musiman
- **Desember:** Revenue tertinggi (47K USD) - efek belanja akhir tahun
- **September:** Revenue terendah (24K USD) - perlu strategi promosi

### 2. Top 3 Produk Terlaris
1. **Tunic** - $17.3K revenue
2. **Jeans** - $13.1K revenue
3. **Pajamas** - $12.8K revenue

### 3. Payment Behavior
- **Credit Card** users spend **5% more** on average ($160.37) vs Cash ($152.70)
- Review rating sama untuk kedua metode (3.0/5.0)

## Business Recommendations

1. **Inventory Management:** Tingkatkan stok Tunic & Jeans menjelang Q4 (Oktober-Desember)
2. **Promotional Strategy:** Buat "September Sale" untuk boost penjualan di bulan terendah
3. **Payment Strategy:** Dorong penggunaan Credit Card/Paylater dengan cashback untuk tingkatkan AOV
